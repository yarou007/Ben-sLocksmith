This template is a collection worksheet, not publishable proof. Do not add it to a page or schema until an authorized owner approves the facts, images, client privacy, location precision, and wording. The machine-readable empty collections and field model live in `data/proof-content.json`.

## Images

For each approved image record the source file, descriptive filename, intrinsic width and height, factual alt text, visible caption, ownership/permission source, and whether the client identity and exact location may be published. Remove people, access credentials, documents, addresses, alarm panels, key codes, and other sensitive building-security details unless specifically approved.

## Publication approval

- [ ] Work record supports the problem, diagnosis, work, and outcome.
- [ ] Client type and general area are accurate.
- [ ] Client identity is omitted or explicitly approved.
- [ ] Location precision is omitted or explicitly approved.
- [ ] Every image is owned/licensed and approved.
- [ ] No review, rating, response-time, license, insurance, certification, brand-authorization, or code-compliance claim has been inferred.
- [ ] `approvedForPublication` is set only after approver, source, and approval date are recorded.
